# GasFree Token

## Pre-requisites

### Node.js

Firebase CLI v11.24.1 is incompatible with Node.js v16.0.0 Please upgrade Node.js to version ^14.18.0 || >=16.4.0

### Firebase CLI

```
npm install -g firebase-tools
```

```
firebase login
```

## Local Development

### Install dependencies

```
npm install
```

### Run the app

```
npm run serve
```
